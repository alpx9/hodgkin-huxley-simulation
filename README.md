# hodgkin-huxley-simulation
A pure Python implementation of the classic Hodgkin-Huxley spiking neuron model. (Klasik Hodgkin-Huxley nöron modelinin Python ile simülasyonu.)

# Hodgkin-Huxley Model Simulation in Python
Bu proje, sinir hücrelerindeki aksiyon potansiyelini tanımlayan ve Nobel ödülü kazandıran Hodgkin-Huxley modelinin (1952) temel bir Python simülasyonudur. Sistem diferansiyel denklemler kullanılarak çözülmüş ve hücre zarındaki voltaj değişimleri görselleştirilmiştir.

Kullanılan Kütüphaneler: > * numpy (Matematiksel işlemler)

scipy (Diferansiyel denklem çözümü için odeint)

matplotlib (Veri görselleştirme)

Nasıl Çalıştırılır?

Gerekli kütüphaneleri terminalinizden kurun: pip install numpy scipy matplotlib

Python dosyasını çalıştırın: python hh_simulation.py
